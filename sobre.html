Sobre o projeto

Projeto Final

Projeto final do curso de progamador front-end ministrafo pela FIRJAN SENAI - Maracanã RJ, referente a avaliação da unidade final de intregração de competências. Entre as exigências da avaliação estão a criação de um CRUD (create, read, update, delete).

Monstro.json

Para atender as necessidades do projeto foi buscada uma api sobre o jogo Monster Hunter World da CAPCOM, devido a falta de alguns itens na api com imagens e ícones, foi decido criar uma fake api com o json-server, que atenderia as necessidades do projeto.
